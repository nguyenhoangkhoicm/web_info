Trợ lý thông minh Siri

Siri là trợ lý được phát triển để phục vụ các bạn sinh viên tìm kiếm thông tin liên quan đến học tập và tra cứu lịch học, và bảng điểm của mình.
Hiện tại phần mềm chỉ hỗ trợ trên hệ điều hành Window. Để tải phần mềm các bạn click vào nút tải xuống bên dưới.

Hướng dẫn cài đặt chương trình

Bước 1

Trước khi cài đặt phần mềm, chúng ta cần cài Python phiên bản (3.9.13) về máy. Bạn có thể tải Python tại đây.

Bước 2

Chúng ta sẽ tải một tệp có tên là requirement.txt. Bạn có thể tải tệp này tại đây.

Bước 3

Chúng ta mở thư mục chứa file requirement.txt vừa tải về. Sau đó, mở cửa sổ CMD và gõ lệnh sau:
pip install -r requirement.txt

Bước 4

Sau khi các thư viện được cài đặt thành công, nhấp vào file Siri.exe để tiến hành cài đặt chương trình.

Bước 5

Sau khi cài đặt chương trình thành công, giao diện trợ lý sẽ hiện lên
